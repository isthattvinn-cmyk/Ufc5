# Night Circuit Legends

Night Circuit Legends is a browser racing game prototype with:

- Story mode races that unlock as you win events
- Car progression through speed, acceleration, handling, and nitro upgrades
- Persistent local save data with credits, XP, and win tracking
- A code-room versus prototype for race setup between tabs/windows on the same hosted site

## Controls

- `W` / `Arrow Up`: accelerate
- `S` / `Arrow Down`: brake
- `A` / `D` or arrow keys: steer
- `Shift`: nitro burst
